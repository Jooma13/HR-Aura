# HR Aura

An automation app that helps HR Professionals manage employees, departments, and leave requests.

## Features

- Employee management
- Department management
- Leave request system

## Setup Instructions

1. Ensure Python 3.x is installed.
2. Create a virtual environment: `python -m venv venv`
3. Activate the virtual environment: `venv\Scripts\activate` (on Windows)
4. Install dependencies: `pip install -r requirements.txt`
5. Set up a MySQL database named 'hraura_db'.
6. Update `hraura/settings.py` with your MySQL username and password.
7. Run migrations: `python manage.py makemigrations && python manage.py migrate`
8. Create a superuser: `python manage.py createsuperuser`
9. Run the development server: `python manage.py runserver`
